# Cps-test
Cps test built in python with tkinter
## Modes
* 1 second test
* 5 Second test
* 10 Second test
* Custom setting for a specific time

## requirements
* Install Tkinter with pip in your terminal 
  ```sh
  pip install tk
  ```
  Then run <a href="https://github.com/BSRT00/Cps-test/blob/main/Main.py">Main.py</a> in python
